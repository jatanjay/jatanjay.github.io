    Few of the projects I've worked on

  • Song and Dance Man:
    • An in-depth corpus based analysis on the works of artist Bob Dylan from the year 1960 to 2020. Explores and answers various questions pertaining to Dylan's corpus by using concepts of Lingustics and principles of NLP, ML.
    Uses : Python (nltk, spaCy, numpy, pandas, keras etc.)
  • CorpusGenius:
    • A robust solution to generate lyrical corpus and other meta data such as albums, titles, lyrics etc. for the specified artist using Genius.com's API.
    Uses : Python (numpy, pandas etc.)
  • Homerun !:
    • Predicting trajectory, range, time of flight etc. of a homerun from an actual baseball game. Calculating analytical and theoretical solution and comparing it to the actual data.
    Uses : MATLAB
  • Where's my Ambulance?:
    • A model to mimic sequential traffic signal systems that go on/off based on Dijkstra's shortest path algorithm so as to help an ambulance travel quickly between endpoints in times of crisis.
    Uses : Java, Arduino etc.
  • CoolerPAD:
    • A smart cooling station for laptop that switches on/off once an user defined temprature is reached.
    Uses : Arduino, temprature sensors, fans etc.
  • C to C to C:
    • A signal generator that can generate a square wave of one of the 25 distinct musical notes. The song thus coded can be heard using an inbuilt TRRS jack.
    Uses : C, ATmega328P etc.
  • Visualizing Music:
    • Visualizing Super Mario Bros. theme on an oscilloscope
    Uses : Arduino etc.
